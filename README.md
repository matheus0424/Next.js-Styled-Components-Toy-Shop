## Step

```bash
# Clone este repositório
$ git clone https://github.com/matheus7864/Next.js-Styled-Components-Toy-Shop.git

# Entre na pasta
$ cd Next.js-Styled-Components-Toy-Shop

# Instale as dependências
$ yarn

# Para iniciar o projeto
$ yarn start

# O app vai inicializar em <http://localhost:3000>
```
